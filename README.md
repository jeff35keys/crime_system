# Crime Management System

A web-based system for managing crime records.

## What it does

- Police can add and manage crime records
- Organizations can search and request records
- Police can approve requests
- Organizations can download approved records

## Technology Used

- PHP
- MySQL
- HTML/CSS
- JavaScript

## Setup

1. Import database from `database/crime_system.sql`
2. Copy `db.example.php` to `db.php`    
3. Update `db.php` with your database info
4. Open in browser
